# Niveis de Diretriz

## DEVE
Regra obrigatoria. Descumprimento bloqueia merge, salvo excecao formal documentada.

## DEVERIA
Regra recomendada. Pode haver excecao com justificativa registrada no PR ou ADR.

## PODE
Opcao permitida. Time escolhe conforme contexto, mantendo compatibilidade e seguranca.

## Aplicacao pratica
- Seguranca/compliance: normalmente DEVE.
- Padrao de organizacao interna de codigo: DEVERIA.
- Ferramentas auxiliares sem impacto de risco: PODE.
