# Templates Disponiveis

## Archetypes suportados
- `node-consumer`: baseline inicial para novos repositorios Node consumidores dos workflows centrais.

## Uso recomendado
- Escolher um archetype suportado.
- Copiar os arquivos para o novo repositorio.
- Substituir placeholders.
- Configurar branch protection no GitHub.
- Validar o primeiro PR com smoke test e rollback.
- Nao criar `ISSUE_TEMPLATE` no consumidor por padrao.
